Detailed Description

The OBJD (OBJect Definition) is the main chunk for an object, and is the first chunk loaded by the VM.

Property Documentation

ushort Iffinator.Flash.OBJD.AllowsIntersectionID
get

The ID of a BHAV that contains a function (tree) used as a last resort for intersection calculation (to determine if two objects are allowed to intersect).

ushort Iffinator.Flash.OBJD.BaseGraphicsID
get

If non-zero, the base graphic is the resource ID of the first (or only) DGRP resource associated with this object. If zero, it is not known how the image is provided for the object.

ushort Iffinator.Flash.OBJD.BodystringsID
get

The body strings ID is the ID of a STR# resource that contains various information about the character, including things like the gender, sex, color, body type, and how to dress them for various circumstances. Only characters and NPCs have this set.

int Iffinator.Flash.OBJD.ChunkID
get

The chunk ID of this OBJD chunk.

ushort Iffinator.Flash.OBJD.Disabled
get

Presumably, the disabled value is set to one to prevent the object from showing up in the game. There are some shutters that have this set; no other object does.

ushort Iffinator.Flash.OBJD.GardeningFuncID
get

The ID of the BHAV chunk that contains the code for the object's gardening function. This should be ignored if an OBJf with the same ID as this OBJD is present.

uint Iffinator.Flash.OBJD.GUID
get

A unique 32-bit GUID for this object.

ushort Iffinator.Flash.OBJD.InitialStackSize
get

The initial size to set the stack for this object's code to.

ushort Iffinator.Flash.OBJD.MainFuncID
get

The ID of the BHAV chunk that contains the code for the object's main function. This should be ignored if an OBJf with the same ID as this OBJD is present.

ushort Iffinator.Flash.OBJD.MasterID
get

The master ID for this OBJD. If this is 0, the object is a single-tile object.

ushort Iffinator.Flash.OBJD.NumGraphics
get

The number of graphics is the number of DGRP resources that are used to display various states of the object.

ushort Iffinator.Flash.OBJD.Portal
get

Is this object a portal (staircase)?

ushort Iffinator.Flash.OBJD.SLOTID
get

The slot ID is the ID of the SLOT resource associated with this object. The SLOT resource provides routing infomation that allows a character to approach the object.

ushort Iffinator.Flash.OBJD.SubIndex
get

The sub index of this OBJD. If this is -1, the object is a master for a multi-tile object. In a multi-tile slave OBJD, the sub-index gives the (x,y) offset for the tile. The upper byte is the zero-relative y offset, while the lower byte is the zero-relative x offset.

ushort Iffinator.Flash.OBJD.TreeTableID
get

The tree table ID is the resource ID of a TTAB (i.e., the set of menu actions) that is used to interact with the object. If zero, there is no interaction with the object.
