string Paloma.TargaFooter.Signature
get

This string is formatted exactly as "TRUEVISION-XFILE" (no quotes). If the signature is detected, the file is assumed to be a New TGA format and MAY, therefore, contain the Developer Area and/or the Extension Areas. If the signature is not found, then the file is assumed to be an Original TGA format.
