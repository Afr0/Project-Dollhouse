Detailed Description

Base class for all UIElements. This class has all the common stuff that user interface components have. This includes:

X,Y, ScaleX, ScaleY, Visible, Parent etc.

The UIElement has its own coordinate space. You can imagine that when you draw a texture inside a UIElement you are drawing it at 0,0 within the UIElement. This is then translated by the engine to screen coordinates.

Standard UIElement update method. All sub-classes should call this super method.

The argument is an UpdateState object, this object contains everything you may need during update including GameTime, MouseState, KeyboardState etc.

This is useful because it means we dont ask for Mouse & Keyboard state in every UIElement which would be wasteful
