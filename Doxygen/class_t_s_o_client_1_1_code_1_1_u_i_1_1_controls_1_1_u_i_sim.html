override void TSOClient.Code.UI.Controls.UISim.Update ( UpdateState  state)
virtual

Standard UIElement update method. All sub-classes should call this super method.

The argument is an UpdateState object, this object contains everything you may need during update including GameTime, MouseState, KeyboardState etc.

This is useful because it means we dont ask for Mouse & Keyboard state in every UIElement which would be wasteful
