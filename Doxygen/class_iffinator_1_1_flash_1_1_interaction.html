short Iffinator.Flash.Interaction.ActionFuncID

A 2-byte chunk ID referring to the BHAV subroutine executed when the interaction is selected.

uint Iffinator.Flash.Interaction.AttenuationCode

(added in version 7) - A 4-byte unsigned integer that controls the attenuation value described below. The possible values for this field are 0 to 4, which correspond to custom (specified in the next field), none, low, moderate and high respectively. Presumably, the attenuation values of these constants are 0, 0.02, 0.1 and 0.2, which are the most commonly used values in TTAB versions predating this field.

float Iffinator.Flash.Interaction.AttenuationValue

A 4-byte IEEE 754 float. If the attenuation code is non-zero, this field is usually zero/garbage, otherwise it specifies how quickly the interaction advertisement fades.

uint Iffinator.Flash.Interaction.AutonomyThreshold

A 4-byte unsigned integer, usually 50 or 100. Only Sims whose autonomy level is higher than or equal to the threshold are able to perform this interaction autonomously.

short Iffinator.Flash.Interaction.GuardFuncID
